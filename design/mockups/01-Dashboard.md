# 01 - Dashboard

## Mockup
- Sol: Sidebar
- Ust: Topbar
- Ana icerik:
  - KPI kartlari satiri (Toplam Ogrenci, Aktif Program, Tamamlanan Ders)
  - Orta sol: Son aktiviteler listesi
  - Orta sag: Haftalik ozet paneli
  - Alt: Hedef tamamlama ilerleme alani

## Kullanici Akisi
1. Kullanici dashboard ekranini acar.
2. KPI kartlari ile genel durumu gorur.
3. Son aktivitelerden ilgili kayda gider.
4. Haftalik ozetten detay ekrana yonlenir.

## Alanlar
- Toplam Ogrenci
- Aktif Program
- Tamamlanan Ders
- Son Aktivite listesi
- Haftalik ozet metrikleri
- Hedef tamamlama yuzdesi

## Butonlar
- Yeni Ogrenci
- Haftalik Program
- Detaya Git

## Renkler
- Arka plan: Acik nötr
- Yuzey kartlari: Beyaz
- Primary vurgu: Mavi ton
- Success: Yesil ton
- Warning: Turuncu ton
- Info/Secondary: Turkuaz ton

## Durumlar
- Veri yukleniyor
- Veri var
- Veri yok (empty state)
- Hata (baglanti/veri)
- Buton disabled
