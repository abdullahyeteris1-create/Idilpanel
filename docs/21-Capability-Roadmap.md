# Capability Roadmap

Date: 2026-06-29
Approach: Capability-driven delivery

## Capability 1

Name: Complete Lesson
Status: Planning
User Value: Kullanici bir dersi olusturur, olcum verisini girer, kaydi tamamlar.

## Capability 2

Name: Measure Reading
Status: Backlog
User Value: Okuma performansi metrikleri (WPM, anlama, gelisim) olusur.

## Capability 3

Name: Generate Reports
Status: Backlog
User Value: Ogrenci ve veli icin anlamli rapor ciktilari uretilir.

## Capability 4

Name: Dashboard Analytics
Status: Backlog
User Value: Karar destek metrikleri dashboard uzerinden izlenir.

## Capability 5

Name: Parent Reporting
Status: Backlog
User Value: Veliye yonelik paylasilabilir ilerleme ozeti sunulur.

---

## Sequencing Rule

- Capability 1 tamamlanmadan Capability 2 aktif implementasyona gecmez.
- Her capability DoR ile baslar ve DoD ile kapanir.
- Her capability kapanisinda E2E dogrulama zorunludur.

## Progress States

- Planning
- In Progress
- Review
- Closed
